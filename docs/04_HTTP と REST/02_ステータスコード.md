# ステータスコード

ステータスコードは、リクエストの結果をクライアントに伝える番号です。

| コード | 意味 |
| --- | --- |
| `200 OK` | 成功 |
| `201 Created` | 作成成功 |
| `204 No Content` | 成功したが本文なし |
| `400 Bad Request` | リクエストが不正 |
| `401 Unauthorized` | 認証が必要 |
| `403 Forbidden` | 権限がない |
| `404 Not Found` | 見つからない |
| `500 Internal Server Error` | サーバーエラー |

Web API では、レスポンス本文だけでなく **適切なステータスコード** を返すことが重要です。
