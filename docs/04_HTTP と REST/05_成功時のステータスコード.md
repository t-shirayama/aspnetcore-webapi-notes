# 成功時のステータスコード

成功時のステータスコードは、処理の結果をクライアントに伝えるために使います。

よく使うものは次の通りです。

| コード | 使う場面 |
| --- | --- |
| `200 OK` | 取得や更新に成功し、レスポンス本文を返す |
| `201 Created` | 新しいリソースの作成に成功した |
| `204 No Content` | 成功したが、返す本文がない |

作成成功では、基本的に **`201 Created` を返す** と意図が明確です。作成されたリソースの URL を `Location` ヘッダーで返すと、さらに親切です。

```csharp
return Results.Created($"/articles/{article.Id}", response);
```

削除成功では、本文が不要なら `204 No Content` がよく使われます。

```csharp
return Results.NoContent();
```
