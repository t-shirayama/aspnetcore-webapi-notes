# レスポンス型の明示

API ドキュメントを分かりやすくするには、返す型やステータスコードを明示します。

Controller では `[ProducesResponseType]` を使えます。

```csharp
[ProducesResponseType(typeof(ArticleResponse), StatusCodes.Status200OK)]
[ProducesResponseType(StatusCodes.Status404NotFound)]
public IActionResult Get(int id)
{
    return Ok();
}
```

Minimal API では `Produces` や Typed Results を使うと、OpenAPI に情報を反映しやすくなります。
