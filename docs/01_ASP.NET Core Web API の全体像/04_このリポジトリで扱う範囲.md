# このリポジトリで扱う範囲

このリポジトリでは、ASP.NET Core Web API を作るときに必要になる基本要素を章ごとに整理します。

- プロジェクト作成と `Program.cs`
- HTTP、REST、ルーティング
- リクエスト、レスポンス、バリデーション
- DTO、Entity、過剰投稿、Mass Assignment
- 例外処理、ProblemDetails
- DI、設定、ミドルウェア
- EF Core、データアクセス
- 認証、認可
- セキュリティ、秘密情報、認可チェック漏れ
- テスト戦略
- Controller、ActionResult、Model Binding
- OpenAPI、API ドキュメント
- 運用、相関 ID、監視、設定管理

まずは **API として外から見える動き** を押さえ、その後に ASP.NET Core の仕組みを学ぶ順番にしています。
