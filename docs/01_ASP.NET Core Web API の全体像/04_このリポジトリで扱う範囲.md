# このリポジトリで扱う範囲

このリポジトリでは、ASP.NET Core Web API を作るときに必要になる基本要素を章ごとに整理します。

- プロジェクト作成と `Program.cs`
- HTTP、REST、API 設計、ルーティング
- Controller、ActionResult、リクエスト、レスポンス
- DTO、Entity、Model Binding、Mass Assignment
- バリデーション、例外処理、ProblemDetails
- DI、設定、ミドルウェア
- EF Core、データアクセス
- 認証、認可、セキュリティ
- テスト戦略、ログ、運用、OpenAPI

まずは **API として外から見える動き** を押さえ、その後に ASP.NET Core の仕組みを学ぶ順番にしています。
