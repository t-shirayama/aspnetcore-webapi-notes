# リクエストボディ

`POST` や `PUT` では、作成や更新に必要なデータをリクエストボディで送ります。

```csharp
public record CreateArticleRequest(string Title, string Body);

app.MapPost("/articles", (CreateArticleRequest request) =>
{
    return Results.Created($"/articles/1", request);
});
```

ASP.NET Core は JSON の本文を C# の型に変換します。

入力用の型は、Entity とは分けて **リクエスト DTO** として作ると扱いやすくなります。
