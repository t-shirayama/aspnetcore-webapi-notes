# レスポンスを返す

Minimal API では `Results.Ok`、`Results.Created`、`Results.NotFound` などでレスポンスを返せます。

```csharp
app.MapGet("/articles/{id}", (int id) =>
{
    if (id <= 0)
    {
        return Results.NotFound();
    }

    return Results.Ok(new { id, title = "Sample" });
});
```

Controller では `Ok()`、`CreatedAtAction()`、`NotFound()` などを使います。

レスポンスは **データとステータスコードをセット** で考えます。
