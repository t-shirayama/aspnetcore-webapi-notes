# ソートと検索

一覧 API では、ソートと検索条件をクエリ文字列で表すことが多いです。

```text
GET /articles?keyword=api&sort=-createdAt
GET /articles?status=published&category=dotnet
```

`sort=-createdAt` のように、先頭の `-` で降順を表す設計もあります。

検索条件を増やすときは、次の点を考えます。

- インデックスが効くか
- 大文字小文字を区別するか
- 部分一致か完全一致か
- 複数条件は AND か OR か
- ソート対象に許可リストを設けるか

特にソート項目をそのまま SQL に埋め込むと危険です。許可された項目だけを受け付けるようにします。

```csharp
var query = sort switch
{
    "createdAt" => articles.OrderBy(article => article.CreatedAt),
    "-createdAt" => articles.OrderByDescending(article => article.CreatedAt),
    _ => articles.OrderBy(article => article.Id)
};
```
