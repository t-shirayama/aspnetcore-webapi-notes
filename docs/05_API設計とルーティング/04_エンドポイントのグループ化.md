# エンドポイントのグループ化

Minimal API では `MapGroup` を使って、共通のプレフィックスを持つエンドポイントをまとめられます。

```csharp
var articles = app.MapGroup("/articles");

articles.MapGet("/", () => Results.Ok());
articles.MapGet("/{id}", (int id) => Results.Ok(id));
articles.MapPost("/", (CreateArticleRequest request) => Results.Created());
```

グループ化すると、URL の重複を減らし、認可やタグ付けなどの共通設定もしやすくなります。
