# 遅延評価

LINQ の問い合わせは、書いた時点ではすぐに実行されないことがあります。

```csharp
var query = db.Articles.Where(article => article.Published);

var articles = await query.ToListAsync();
```

この例では、`ToListAsync` を呼んだタイミングで SQL が実行されます。

`IQueryable<T>` は DB に問い合わせる前の式、`List<T>` は実行後の結果です。

遅延評価を理解していないと、不要なデータをメモリに読み込んだり、想定より多く SQL を発行したりします。
