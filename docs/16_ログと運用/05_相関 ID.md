# 相関 ID

相関 ID は、1 つのリクエストや一連の処理を追跡するための ID です。

複数のサービスやログにまたがって調査するときに役立ちます。

```text
RequestId: 0HN...
TraceId: 00-...
CorrelationId: 7f3...
```

API でエラーが起きたとき、レスポンスに `traceId` を含めると、利用者から問い合わせを受けたときにログを探しやすくなります。

```json
{
  "title": "An unexpected error occurred.",
  "status": 500,
  "traceId": "00-..."
}
```

相関 ID は、ログ、エラーレスポンス、分散トレーシングをつなぐ手がかりです。
