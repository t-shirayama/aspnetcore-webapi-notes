# タイムアウト

タイムアウトは、処理が終わらないまま待ち続けることを防ぐための設定です。

外部 API、DB、HTTP クライアント、バックグラウンド処理では、無期限に待たないようにします。

```csharp
builder.Services.AddHttpClient("external-api", client =>
{
    client.Timeout = TimeSpan.FromSeconds(10);
});
```

タイムアウトが短すぎると正常な処理も失敗します。長すぎると障害時にスレッドや接続を使い続け、他のリクエストにも影響します。

## よくある失敗

- 外部 API の応答を無期限に待つ
- DB が遅いときに API 全体が詰まる
- クライアント、API、DB のタイムアウト方針がばらばらになる
