# サービス登録

ASP.NET Core では、アプリで使う部品を DI コンテナーに登録します。

```csharp
builder.Services.AddScoped<ArticleService>();
builder.Services.AddControllers();
```

登録したサービスは、Controller や Minimal API の引数、コンストラクターから受け取れます。

```csharp
app.MapGet("/articles", (ArticleService service) => service.GetAll());
```

サービス登録は **アプリ全体で使う依存関係の一覧** です。
