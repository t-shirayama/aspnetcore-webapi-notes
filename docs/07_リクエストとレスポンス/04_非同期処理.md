# 非同期処理

Web API では、データベースアクセスや外部 API 呼び出しなどの待ち時間が発生します。

そのような処理では `async` と `await` を使います。

```csharp
app.MapGet("/articles", async (ArticleDbContext db) =>
{
    var articles = await db.Articles.ToListAsync();
    return Results.Ok(articles);
});
```

非同期処理は、待っている間にスレッドを占有し続けないため、Web アプリでは重要です。
