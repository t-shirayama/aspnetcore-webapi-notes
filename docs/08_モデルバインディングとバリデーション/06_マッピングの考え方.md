# マッピングの考え方

マッピングは、Entity と DTO を変換する処理です。

```csharp
static ArticleResponse ToResponse(Article article)
{
    return new ArticleResponse(
        article.Id,
        article.Title,
        article.CreatedAt);
}
```

小さな API では、手書きのマッピングで十分なことが多いです。変換が多く複雑になってきたら、AutoMapper などのライブラリを検討します。

AutoMapper は便利ですが、暗黙的な変換が増えると、どの項目がどこで設定されるか追いにくくなる場合があります。

実務では、**単純なら手書き、量が増えて規則的ならライブラリ** という判断が現実的です。
